# Myproject (벽돌깨기 게임)

# 1. 소개
간단한 게임 프로젝트 구상 중에 벽돌깨기라는 게임이 눈에 들어왔고, 별다른 큰 작업없이 만들 수 있다는 점이 맘에 들어 이 게임으로 선정했습니다

# 2. 게임구현
**게임시작을 누르기전에 easy, normal, hard 등 난이도 선택 후 난이도에 따른 한 스테이지를 구현한다(최종결정)**
